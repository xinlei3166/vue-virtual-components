# vue-virtual-components
Vue3 虚拟组件

## Overview
随着前端功能的日益负责和数据量的增加，部分UI组件库不支持数据量过大的功能。
这个库主要是开发这些常用的虚拟组件。组件和常用的UI组件库风格样式一致，也可以自己修改样式。
